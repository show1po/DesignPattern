# Java Framework  #

## 緩存框架 ##

本機緩存：
----

### OSCache ###
 
* 緩存任何 Java Object 。
* 允許永久緩存，寫入硬碟。
* 支援群集，緩存紀錄過期可插入式刷新。

### JCache ###

javax.cache

1. 原子操作，如同 java.util.concurrentmap
2. 從緩存中讀取
3. 寫入緩存
4. 緩存事件監聽器
5. 數據統計
6. 隔離級別事務
7. 緩存註解
8. 保存 key-value 類型的泛型緩存。
9. 引用保存 堆緩存(?


### Guava Cache ###

全內存的本機緩存，提供執行緒安全實現機制。

分布式緩存
----

### EhCache ###

將資料保存於不同服務器內存中，減輕負載。
通過聲明配置、 XML 配置、程序中配置、調用建構子中傳入參數。

### Jboss cache ###

Java 事務處理緩存系統，服務器應用與集群解決方案。

### Redis ###

Key-value 高性能資料庫系統，提供多語言 API 。基於內存，持久日誌型。
